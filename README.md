# Giftastic

## Overview
Giftastic is a frontend web application that allows users to search and interact with GIFs using the Giphy API. Users can generate GIFs based on predefined or custom topics and toggle animations on click.

This project demonstrates API integration, dynamic DOM manipulation, and event-driven user interactions in JavaScript.

## Live Demo
Try it here: https://giftastic-six.vercel.app

## Features
- Fetch GIFs from the Giphy API based on user-selected topics
- Dynamically generate buttons for new search terms
- Toggle GIFs between still and animated states on click
- Display multiple results with ratings

## Tech Stack
- JavaScript (Vanilla)
- HTML / CSS
- Giphy API
- jQuery (if used)

## How to Use
1. Open the live demo or run locally via `index.html`
2. Click an existing topic button or add a new one
3. View GIF results displayed on the page
4. Click a GIF to toggle between still and animated versions

## Example Functionality
- User clicks a topic → GIFs load dynamically
- User clicks a GIF → animation toggles on/off
- User adds a topic → new button appears instantly

## Purpose
This project demonstrates early experience with API integration, DOM manipulation, and building interactive frontend applications.
